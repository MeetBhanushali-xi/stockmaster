This is a website about the problem statement given to us by ODOO in the SPIT Hackathon.
We the team of 4 People Hriday Dedhia, Meet Bhanushali, Bhavya Dharod, Vrushabh Mehta have succesfully solved the problem statement.


StockMaster is a beginner-friendly, hackathon-ready Inventory Management System built using a simple and clean architecture.
It uses Node.js + Express for the backend and HTML + Bootstrap 5 + JavaScript for the frontend.
All data is stored in JSON files instead of a database, making it extremely easy to set up, run, and modify.

Features
Authentication
Login page with credential validation
User details stored in users.json
Secure API-based login using Express

Dashboard
Overview of inventory metrics
Built-in KPIs:
Total Products
Low Stock Items
Pending Receipts
Pending Deliveries
Internal Transfers (Scheduled)
Fully responsive Bootstrap design

Products Module (CRUD)
Add New Product
Edit Product
View Product List
Auto-generated Product ID
Responsive table UI
Data saved inside products.json

REST APIs:
GET /api/products
POST /api/products
PUT /api/products/:id

Technology Stack
Frontend
HTML5
CSS / Bootstrap 5
Vanilla JavaScript
REST API Integration using fetch()
Backend
Node.js
Express.js
CORS

File System (fs) for JSON storage
Modular route-based API design
Storage
JSON Files (no database needed)
